

# Ingenium 2026 — Chitkara University Tech Festival Website

A modern, dark-themed, high-conversion frontend website for a 3-day tech fest (March 13–15, 2026), inspired by [cognizance.org.in](https://www.cognizance.org.in/).

---

## Design Direction
- **Dark theme** with glowing neon accents (electric blue/purple gradients)
- Bold typography, smooth scroll animations, and particle/tech-style visual effects
- Fully responsive (mobile, tablet, desktop)
- No backend — purely static frontend showcase

---

## Pages & Sections

### 1. Landing Page (Single-page scroll)

**Hero Section**
- Full-screen dark background with animated gradient/particle effects
- "Chitkara University presents…" subtitle
- "MARCH 13 – 15, 2026" date badge
- Large fest name **"INGENIUM"** with a glowing tagline (e.g., "Where Innovation Meets Engineering")
- Two CTA buttons: "Register Now" and "Explore Events" (linking to sections/pages)

**About Section**
- Brief intro about Chitkara University and the fest
- Key stats (e.g., "5000+ Participants", "50+ Events", "3 Days of Innovation")
- Animated counters on scroll

**Past Performers / Highlights**
- Horizontal scrolling carousel of past performers/guests with image cards
- Each card shows name, role (Speaker / Artist / Comedian)
- Placeholder images and names for now

**Events Section**
- Featured event cards with hover effects and descriptions
- Categories like Robotics, Coding, Hackathons, Design, etc.
- "Explore All Events" button linking to Events page
- Prize pool callouts

**Sponsors / Partners**
- Logo marquee/carousel of sponsor logos (placeholder logos)
- "Previous Workshop Partners" style like Cognizance

**Schedule Overview**
- 3-day tabbed schedule (Day 1 / Day 2 / Day 3)
- Timeline-style layout showing events per time slot

**FAQs**
- Accordion-style FAQ section with common questions about the fest

**CTA / Book Now**
- Bold call-to-action banner ("Book Your Slots Now") with a button
- Futuristic robot/tech illustration alongside

**Footer**
- About text, social media links (Instagram, Facebook, Twitter, LinkedIn)
- Quick links: About, Events, Workshops, Gallery, Contact
- Address: Jhansla, Punjab
- Terms & Conditions, Privacy Policy links

### 2. Events Page
- Grid of all event cards with filters by category
- Each card: event name, short description, prize pool, image

### 3. Gallery Page
- Photo grid/masonry layout with placeholder images from past fests

### 4. Contact Page
- Contact form (name, email, message — frontend only, no submission)
- Map embed showing Chitkara University, Jhansla, Punjab
- Social media links

---

## Navigation
- Sticky transparent navbar that becomes solid on scroll
- Links: Home, Events, Gallery, Contact
- "Register" CTA button in navbar
